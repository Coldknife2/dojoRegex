Le mot de la fin


Savoir se servir de quelque chose c'est aussi savoir quand ne pas s'en servir

On vous a fait pratiquer les Regex pour l'exercice

Maintenant, ce n'est normalement pas très adapté pour plusieurs cas d'usages

Par exemple, pour vérifier qu'un input est un nombre (saisie textuelle) il est préférable de se servir de sa librairie standard

Pour ne citer que le js : Number.isInteger pour les nombres entier par exemple
