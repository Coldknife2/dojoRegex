Mise en situation

On a ces trois PDF dans un dossier,
avec à la fin le jour, mois puis année
``` BUL0064137631032018.pdf BUL0064137631072019.pdf BUL0064137631032019.pdf ```
Je veux garder l'année et le mois, en format américain (yyyy-mm) pour que cela se trie tout seul par nom
``` ls | sed 's/BUL..........\(..\)\(....\)..../"\0" "\2-\1.pdf"/' | xargs -n 2 mv ```
J'aurais pu faire :
``` ls | sed 's/BUL.\{10\}\(.\{2\}\)\(.\{4\}\)/"\0" "\2-\1.pdf"' | xargs -n 2 mv ```
Mes pdfs ont étés renommés

``` 2018-03.pdf 2019-07.pdf 2019-03.pdf ```
